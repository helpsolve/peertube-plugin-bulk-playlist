# PeerTube Plugin: Bulk Playlist

**Version:** 1.0.4  
**Author:** helpsolve  
**License:** AGPL-3.0

---

## Description

This PeerTube plugin allows users to **bulk add selected videos** to an **existing or new playlist** directly from the My Videos page. It is fully client-side and works with PeerTube v8.0.0 or higher.

---

## Usage

1. Go to **My Videos** in your PeerTube instance.  
2. Select multiple videos.  
3. Click the **“Add selected to playlist”** action.  
4. Choose an **existing playlist** or **create a new one**.  
5. Set privacy if creating a new playlist.  
6. Click **Add** — your videos will be added in bulk.

---

## Requirements

- PeerTube **v8.0.0 or higher**  
- Node.js and npm installed  
- The plugin is **client-side only**; no extra server setup required
